# Pathogen Genomics Workshop: Pre/Post-Test

This assessment is designed to evaluate your knowledge of pathogen genomics workflows, from sequencing technologies to phylogenetic analysis.

**Total Questions**: 20
**Time Allotted**: 30 Minutes

---

## Multiple Choice Questions

**1. Which sequencing technology is known for producing "long reads" (thousands of base pairs) but typically has higher error rates than Illumina?**
- A) Illumina
- B) Sanger Sequencing
- C) Oxford Nanopore (ONT)
- D) Ion Torrent

**2. In the context of the Central Dogma of biology, what is the correct flow of genetic information?**
- A) DNA -> RNA -> Protein
- B) Protein -> RNA -> DNA
- C) RNA -> DNA -> Protein
- D) DNA -> Protein -> RNA

**3. Which file format is specifically used to store raw sequencing reads along with their quality scores?**
- A) FASTA
- B) FASTQ
- C) BAM
- D) GFF

**4. In the Galaxy interface, where can you find all the tools available for data analysis?**
- A) The Right Panel (History)
- B) The Middle Panel (Main)
- C) The Left Panel (Tools)
- D) The Top Bar (User)

**5. A Phred quality score of 30 (Q30) corresponds to what percentage of base calling accuracy?**
- A) 90%
- B) 99%
- C) 99.9%
- D) 99.99%

**6. What is the primary purpose of using 'Trimmomatic' or 'Cutadapt' during NGS data analysis?**
- A) To assemble reads into contigs
- B) To remove low-quality bases and adapter sequences
- C) To annotate genes in a genome
- D) To visualize the read alignment

**7. Which tool is commonly used to generate a comprehensive quality control report for high-throughput sequencing data?**
- A) SPAdes
- B) Prokka
- C) FastQC
- D) BWA-MEM

**8. What does "Paired-end" sequencing refer to?**
- A) Sequencing both ends of a DNA fragment
- B) Sequencing two different samples at once
- C) Sequencing DNA and RNA together
- D) Sequencing using two different platforms

**9. Which process involves reconstructing a genome from short DNA reads without the use of a reference genome?**
- A) Reference Mapping
- B) Variant Calling
- C) De Novo Assembly
- D) RNA-Seq

**10. The tool 'QUAST' is used during genome assembly to:**
- A) Predict genes and proteins (Annotation)
- B) Evaluate the quality and metrics of the assembly (N50, total length, etc.)
- C) Remove adapter sequences from raw reads
- D) Align reads to a reference genome

**11. Which assembly tool is specifically designed for bacterial genomes and can perform hybrid assembly using both short and long reads?**
- A) Bowtie2
- B) Unicycler
- C) Samtools
- D) MultiQC

**12. The software 'Prokka' is used for:**
- A) Mapping reads to a reference
- B) Rapid prokaryotic genome annotation
- C) Calculating phylogenetic trees
- D) Detecting AMR genes in raw reads

**13. Antimicrobial resistance that occurs due to changes in the existing genome of a bacterium (e.g., chromosomal mutations) is called:**
- A) Acquired resistance
- B) Horizontal gene transfer
- C) Mutational resistance
- D) Plasmid-mediated resistance

**14. Which of the following is a widely used database for identifying antimicrobial resistance (AMR) genes in genomic data?**
- A) GenBank
- B) CARD (The Comprehensive Antibiotic Resistance Database)
- C) UniProt
- D) PDB (Protein Data Bank)

**15. The tool 'ABRicate' or 'StarAMR' is typically used to:**
- A) Build a phylogenetic tree
- B) Predict AMR genes from assembled contigs or raw reads
- C) Trim adapter sequences
- D) Visualize mapping results

**16. In AMR detection, what does 'acquired resistance' usually refer to?**
- A) Resistance gained through spontaneous mutations in core genes
- B) Resistance gained through horizontal gene transfer (e.g., via plasmids)
- C) Natural resistance found in all members of a species
- D) Resistance that is only temporary

**17. Which method is more suitable for high-resolution tracking of outbreaks and identifying small differences between closely related isolates?**
- A) MLST (Multi-Locus Sequence Typing)
- B) SNP (Single Nucleotide Polymorphism) analysis
- C) Gram staining
- D) PCR

**18. A 'Core Genome' describes:**
- A) All genes present in at least one isolate of a species
- B) All genes shared by all (or nearly all) isolates of a species
- C) Only the genes involved in antibiotic resistance
- D) The non-coding regions of a genome

**19. The tool 'Roary' is used to analyze:**
- A) High-quality short reads
- B) The Pangenome (identifying core and accessory genes)
- C) Phred scores across multiple samples
- D) Real-time sequence data from Nanopore

**20. Which web-based tool allows for the visualization of phylogenetic trees alongside epidemiological metadata (e.g., location, time)?**
- A) IGV (Integrative Genomics Viewer)
- B) Microreact
- C) FastQC
- D) SPAdes
